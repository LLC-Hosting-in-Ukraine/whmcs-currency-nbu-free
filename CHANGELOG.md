# Changelog

## v1.0.0 – Initial Release

* Initial public release of WHMCS Currency Auto Update (NBU) – Free
* Automatic daily currency rate updates via WHMCS `DailyCronJob`
* Uses official exchange rates from the National Bank of Ukraine (NBU)
* No API key required
* Safe update logic without recalculating existing invoices
* Lightweight hook-based installation (no admin UI, no database changes)
